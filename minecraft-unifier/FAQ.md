# 我的世界统一器 - FAQ（常见问题解答）

本文档回答了用户最常见的问题。

## 一般问题

### Q1: 什么是"我的世界统一器"？

**A**: "我的世界统一器"是一个跨平台的Minecraft模组兼容工具，旨在打破Java版、基岩版（国际版）、网易中国版之间的资源壁垒，实现基岩版对Java版光影包、各类模组、资源包的完整兼容。

### Q2: 这个项目是官方的吗？

**A**: 不是。这是一个由社区驱动的开源项目，与Mojang Studios或网易游戏没有任何官方关系。

### Q3: 这个项目免费吗？

**A**: 是的，完全免费且开源。我们采用MIT开源协议，您可以自由使用、修改和分发。

### Q4: 我需要付费使用吗？

**A**: 不需要。项目完全免费，但欢迎您通过GitHub Sponsors或其他方式赞助我们，以支持项目的持续开发。

## 使用问题

### Q5: 我需要什么系统才能使用？

**A**: 目前支持以下平台：
- Windows 10/11 (64位)
- Linux (主流发行版)
- Android 5.0+ (ARM64)

### Q6: 我需要什么版本的Minecraft？

**A**: 
- 基岩版（国际版）：最新版本
- 网易中国版：最新版本
- Java版：用于转换模组

### Q7: 我需要正版游戏吗？

**A**: 是的。使用本工具需要您已合法拥有《我的世界》正版游戏。我们不支持盗版游戏。

### Q8: 如何开始使用？

**A**: 请参考 [README.md](README.md) 中的"快速开始"部分，或查看 [TECHNICAL_DOCUMENTATION.md](TECHNICAL_DOCUMENTATION.md) 中的详细使用说明。

## 技术问题

### Q9: 注入后游戏无法启动？

**A**: 请检查以下几点：
1. 确保游戏文件是正版
2. 检查注入器是否正确执行
3. 查看日志文件获取详细错误信息
4. 确保兼容模块已正确放置
5. 尝试重新注入

### Q10: 光影包无法加载？

**A**: 请检查：
1. 光影包是否为Java版格式
2. GLSL着色器是否正确转换
3. Render Dragon API是否正确初始化
4. 查看着色器缓存是否有错误
5. 确认光影包与游戏版本兼容

### Q11: Java模组无法运行？

**A**: 请检查：
1. JVM是否正确初始化
2. API映射是否正确配置
3. 模组是否正确加载
4. 查看Java日志获取详细错误信息
5. 确认模组与游戏版本兼容

### Q12: 网易模组无法运行？

**A**: 请检查：
1. Python解释器是否正确初始化
2. SDK模拟层是否正确注册
3. 模组脚本是否正确转换
4. 查看Python日志获取详细错误信息
5. 确认模组与游戏版本兼容

### Q13: 资源包无法加载？

**A**: 请检查：
1. 文件系统Hook是否正确安装
2. 重定向规则是否正确配置
3. 资源格式是否正确转换
4. 查看文件访问日志
5. 确认资源包与游戏版本兼容

## 兼容性问题

### Q14: 支持哪些光影包？

**A**: 目前支持大部分Java版光影包，包括：
- SEUS系列
- Chocapic13系列
- Sildur's Vibrant系列
- BSL系列

部分高级光影包可能无法完全兼容。

### Q15: 支持哪些模组？

**A**: 目前支持：
- Java版Forge/Fabric模组（部分）
- 网易版Python模组
- 基岩版Add-ons

模组兼容性取决于API映射的完整性。

### Q16: 会影响游戏性能吗？

**A**: 可能会有一定影响，具体取决于：
- 加载的模组数量
- 光影包的复杂度
- 设备性能

我们持续优化性能，建议使用高性能设备。

## 安全问题

### Q17: 使用这个工具安全吗？

**A**: 是的。我们的代码完全开源，接受社区审查。我们：
- 不收集用户数据
- 不进行网络通信
- 不包含恶意代码
- 采用静态注入技术

### Q18: 会被封号吗？

**A**: 理论上不会，因为：
- 我们不修改游戏逻辑
- 不绕过反作弊系统
- 不提供作弊功能

但使用任何第三方工具都有一定风险，请自行承担。

### Q19: 会损坏我的游戏文件吗？

**A**: 我们会自动备份原文件，但建议：
- 修改前手动备份
- 保留原始安装包
- 在测试环境先尝试

## 法律问题

### Q20: 这个工具合法吗？

**A**: 在大多数地区是合法的，因为：
- 我们不分发游戏资产
- 我们不分发受版权保护的模组
- 我们仅提供兼容工具
- 用户需自行拥有正版游戏

但请遵守您所在地区的法律法规。

### Q21: 我可以分发修改后的游戏吗？

**A**: 不可以。修改后的游戏文件包含：
- Mojang Studios的版权资产
- 可能包含模组作者的版权资产

分发这些文件违反版权法。

### Q22: 我可以用于商业用途吗？

**A**: 本工具本身可以用于商业用途（MIT协议），但：
- 不能分发修改后的游戏
- 不能分发受版权保护的模组
- 需遵守游戏服务条款
